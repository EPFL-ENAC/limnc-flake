## Installation

Development version:

```bash
pip install git+https://github.com/EPFL-ENAC/limnc-flake.git
```

Released version:

```bash
pip install git+https://github.com/EPFL-ENAC/limnc-flake.git@1.0.0
```

## Usage

List sub-commands:

```bash
flake --help
```

Show sub-command's options and arguments:

```bash
flake <command> --help
```
